# AI Governance — Real World Cases

Analysis of AI systems currently deployed in the real world,
examined for compliance exposure, governance gaps, and what
mitigation should look like.

These are not hypotheticals. These are live systems affecting
real people today.

## Cases

| # | System | Organization | Primary Issue | Jurisdiction |
|---|--------|-------------|---------------|-------------|
| 001 | [BK Assistant (Patty)](./001-bk-assistant.md/) | Burger King | BIPA voiceprint exposure + minor employee governance gap | Illinois, USA |

## Approach

Each case includes: system description, identified compliance exposure,
affected stakeholders, mitigation strategy, and regulatory trajectory.

Analysis draws on publicly available information only. The goal is
practitioner-level reasoning, not journalism.
